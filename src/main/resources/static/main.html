API usage:

    Directories: /api

    • /customers
    • /policies
    • /claims
    • /payments
    • /coverages

  • POST

    Customer: Long id, String name, List<Policy> policies Policy: Long id, String type, Long price, Long coverageId Claim: Long id, String reason, Long amount, Long policyId, Long coverageId Payment: Long id, Long amount, Long policyId Coverage: Long id, Long limit Example: { "id": 1, "name": "Ivan Ivanov", "policies": [ { "id": 1, "type": "Car insurance", "price": 5000, "coverageId": 1 } ] }

  • GET, PUT, DELETE

    Access any item with /api/{item}/{id}
